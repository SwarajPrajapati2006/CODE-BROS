## 💸 What is $pend Co?

> **$pend Co** is a sleek, web-based **expense management platform** built by Code Bros. It streamlines how companies track, submit, approve, and analyze employee expenses — with a powerful Admin Dashboard and a clean Employee portal.

🔗 **Live Demo:** [bucolic-halva-1aeb9c.netlify.app](https://bucolic-halva-1aeb9c.netlify.app/)

## ✨ Features

### 🔐 Authentication
- User **Login** with email & password
- **Remember Me** functionality
- **Forgot Password** recovery flow
- **Sign Up** for new users

---

### 🛠️ Admin Dashboard
- 📋 **Pending Approvals** — view & act on expenses waiting for review
- 💰 **Expense Analytics** — total submitted, approved, and rejected amounts for the month
- 📊 **Recent Expenses Table** — sortable list with employee, date, category, amount & status
- ✅ **Approve / Reject** individual expense claims with one click
- 🏦 **Corporate Card** management section

---

### ⚙️ Approval Workflow Engine
- **Multi-level Approval Flow** — configure step-by-step approvers (Manager → Finance → Director)
- **Conditional Rules:**
  - Percentage Rule — e.g. if 60% of approvers approve → auto-approved
  - Specific Approver Rule — e.g. if CFO approves → auto-approved
  - Hybrid Rule — combine percentage + specific approver logic
- Add, edit, or remove approval steps and rules dynamically

---

### 👷 Employee Portal
- Submit new expense claims with category, description, amount & receipt upload
- View personal expense history and approval status
- Track which approver is currently reviewing the claim

---

### 🏢 Company Settings
- Set **Company Name** and **Default Currency** (USD, EUR, GBP, JPY)
- Configure **Country** for regional settings

## 🚀 Getting Started

### Run Locally
```bash
# Clone the repository
git clone https://github.com/SwarajPrajapati2006/CODE-BROS.git

# Open in browser
cd CODE-BROS
open index.html
```

> No build tools or dependencies needed — pure HTML, CSS & JavaScript!
